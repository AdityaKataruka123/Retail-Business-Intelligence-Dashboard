# 📊 Retail Business Intelligence Dashboard

## 📌 Project Overview

This project demonstrates an end-to-end Business Intelligence workflow using Python, MySQL, and Power BI.

The project includes:

- Data Cleaning using Python (Pandas)
- SQL Analysis using MySQL
- Interactive Dashboard using Power BI
- Business Insights for decision-making

---

## 🛠 Tools Used

- Python
- Pandas
- NumPy
- MySQL
- Power BI
- Jupyter Notebook

---


## 📈 Dashboard Features

- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Total Customers KPI
- Profit Margin KPI
- Monthly Sales Trend
- Sales by Category
- Sales by Region
- Region & Category Filters
- Executive Insights

---

## 📊 Key Insights

- Total Sales reached **2.33M**
- Technology generated the highest sales.
- West region contributed the highest revenue.
- Profit Margin remained around **12.6%**.
- Monthly sales showed seasonal fluctuations.

### Python

### MySQL

Execute the SQL scripts in the SQL folder.
